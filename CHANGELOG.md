# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [v0.1.1] - 2026-01-06

### Added
- Introduced a game timer and labels for displaying the final score
- Implemented a leaderboard system to track high scores
- Added pause and restart buttons for better game control
- Implemented damage mechanics allowing players to take and deal damage
- Added health display to the HUD

### Fixed
- Resolved various errors in the Heads-Up Display (HUD)

## [v0.1.0] - 2026-01-06

### Added
- Set up the initial project structure and core files
- Added Heads-Up Display (HUD) with enemy despawning on collision
- Enhanced HUD and implemented mob despawning upon collision
- Added a score system to track enemies eaten
- Implemented player movement and enemy spawning mechanics
